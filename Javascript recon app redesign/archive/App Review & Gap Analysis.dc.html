01 · THE PRODUCT

What the app actually does

At its core sits a best-effort static analyzer: point it at JavaScript and it reconstructs the backend API without executing code — tracing fetch / XHR / axios / jQuery / WebSocket calls into endpoints, methods, params, headers, and bodies, then emitting an OpenAPI spec and a recon report. Every finding carries a file:line and a confidence tier; anything dynamic is surfaced, never dropped. The UI wraps this in a multi-target workspace, a capture extension, secret detection, an attack-surface map, and an AI threat-modeling layer.
