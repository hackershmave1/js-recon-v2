ENGINEERING SPEC · BACKEND & PLATFORM

What it takes to run recon at scale

The workspace is I/O- and compute-heavy: it crawls, fetches hundreds of assets, parses ASTs, calls LLMs, and probes live targets — all work that must run off the request thread. This spec defines the async pipeline, queues, data stores, realtime/polling model, and LLM orchestration the product depends on, with traceable requirement IDs for the build.

03 · EXTERNAL ENGINES

The analysis pool is not one Python process

The named engines are separate open-source projects in three languages. Only one (Vespasian) is importable Python; the rest are compiled Go/Rust binaries the workers shell out to and parse — and several make their own outbound requests, so they belong in the scoped egress sandbox, not the internal analyze queue. Queue topology
